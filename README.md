# Dotfiles

All my config files in one place.

## Get Started

1.  Clone this repo down to some directory: `/home/<username>/some/dir/`
2.  Look at the contents of `scripts/gen-symlinks.sh` **BEFORE** running it.
3.  Run `bash scripts/gen-symlinks.sh`

### Tracking New Files

1.  Create the new file in the root directory of this project
2.  Add the file to the `main()` function in `scripts/gen-symlinks.sh`
3.  Run `bash scripts/gen-symlinks.sh`
